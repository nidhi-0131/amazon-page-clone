# Amazon.in Homepage Clone

## 📝 Project Description
This is a **static clone of the Amazon.in homepage** created as part of the Web Development Major Project (Second Semester).

The goal was to replicate the look and feel of the real Amazon India website as closely as possible using HTML, CSS, and JavaScript.

**Reference**: https://www.amazon.in/

---

## ✨ Features Implemented

- Fully Responsive Design (Desktop, Tablet, Mobile)
- Sticky Navigation Bar
- Working Search Bar (with simulation + auto-clear)
- Hero Banner Carousel (sliding promotions)
- 4 Product Sections with hover effects
- Add to Cart functionality (live counter + notifications)
- Clean and modern UI

---

## 🛠️ Technologies Used
- HTML5
- CSS3 (Flexbox + Grid)
- JavaScript (Vanilla)
- Font Awesome Icons

---

## 🚀 How to Run
1. Open `index.html` in your browser
2. No installation needed
